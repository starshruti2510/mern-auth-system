# MERN Secure User Authentication System

## Project Overview

This is a full-stack authentication system built using the MERN stack. It includes secure user registration and login, JWT-based authentication, password hashing with bcrypt, and protected routes.

## Features

* User Signup
* User Login
* JWT Authentication
* Protected Routes
* Password Validation
* Frontend and Backend Integration

## Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB
* JWT
* bcrypt

## How to Run the Project

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```
