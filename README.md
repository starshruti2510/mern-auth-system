# MERN Secure User Authentication System

## Project Overview

This project is on MERN Stack Authentication System.

## Features

* User Signup
* User Login
* JWT Authentication
* Protected Routes
* Password Validation
* Frontend and Backend Integration

## Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB
* JWT
* bcrypt

## How to Run the Project

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```
